## React + Redux + SSR(Server Side Rendering) Example

> Using React with Redux. Routing react-router(v4) and SSR.

### How to start

Require [docker-compose](https://docs.docker.com/compose/).

```sh
docker compose up -d
```

Access to http://127.0.0.1:3000.
